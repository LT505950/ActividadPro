# Guía para Gerentes: Consulta y Gestión de Clientes en Actividad Pro

Esta guía explica cómo utilizar la pantalla de gestión de clientes para Gerentes dentro de la aplicación Actividad Pro, que aparece como el último paso del 'Flujo de captura'.

## Descripción de la Pantalla

Al ingresar a la sección **Clientes** desde el menú principal, se presenta la pantalla **'Registros del gerente'**. Esta interfaz está diseñada para que los gerentes supervisen la actividad de sus asesores y gestionen los siguientes pasos del proceso.

La pantalla se divide en dos pestañas principales:
1.  **Mis Clientes**: Muestra los registros de clientes capturados directamente por ti (el gerente).
2.  **Clientes Asesores**: Es la vista principal y muestra una tabla con todos los registros de clientes capturados por los asesores a tu cargo.

La tabla en la pestaña **'Clientes Asesores'** contiene las siguientes columnas:
- **No. empleado**: Número de empleado del asesor que realizó el registro.
- **Fecha Registro**: Fecha y hora en que se creó el registro.
- **Nombre asesor**: Nombre del asesor responsable.
- **CURP**: CURP del cliente registrado.
- **Apovol, EDR, NRP**: Indicadores de estatus representados por círculos de colores, que muestran el estado de cada etapa del proceso para ese cliente.
- **Notas (...):** Un ícono de tres puntos que abre un menú de acciones para ese registro específico.

En la esquina superior derecha, encontrarás un icono de **Filtros** para buscar y segmentar la información mostrada.

## Pasos a Seguir

1.  **Accede a la Vista de Clientes**: Después de iniciar sesión en Actividad Pro, navega al módulo **'Clientes'** en el menú principal.
2.  **Selecciona la Pestaña Correcta**: La pantalla mostrará por defecto la pestaña **'Clientes Asesores'**. Si necesitas ver tus propios registros, pulsa sobre la pestaña **'Mis Clientes'**.
3.  **Consulta la Información**: Revisa la lista de clientes registrados por tu equipo. Utiliza los indicadores de colores (Apovol, EDR, NRP) para conocer rápidamente el estatus de cada trámite.
4.  **Filtra los Registros (Opcional)**: Si buscas un cliente o asesor específico, pulsa el icono de **Filtros** (embudo) en la esquina superior derecha para acotar la lista.
5.  **Gestiona un Cliente**: Para realizar una acción sobre un registro, localiza al cliente en la lista y pulsa el icono de tres puntos (`...`) al final de la fila. Desde aquí podrás **mandar a realizar Estudios** y consultar las **Notas** asociadas.

**Importante**: La función principal de esta pantalla para el gerente es iniciar el envío de 'Estudios'. El resto de la información es principalmente de consulta para dar seguimiento.

## Posibles Errores y Soluciones

- **La lista de clientes no carga o está vacía**: 
  - **Causa**: Problemas de conexión a internet o sincronización lenta.
  - **Solución**: Asegúrate de tener una conexión estable. Desliza el dedo hacia abajo sobre la lista para forzar la actualización. Si el problema persiste, cierra y vuelve a abrir la aplicación.

- **No puedo 'mandar Estudios' para un cliente**:
  - **Causa**: El cliente no cumple con los requisitos previos de estatus (por ejemplo, el estatus de Apovol o EDR no es el correcto) o podrías no tener los permisos necesarios.
  - **Solución**: Verifica que los círculos de estatus para ese cliente indiquen que está listo para el siguiente paso. Si todo parece correcto, contacta a soporte técnico para revisar tus permisos de usuario.

- **La información de un registro es incorrecta**:
  - **Causa**: Error durante la captura de datos por parte del asesor.
  - **Solución**: Esta pantalla es de consulta y gestión, no de edición. Debes contactar directamente al asesor responsable del registro para que realice la corrección desde el origen.

# Guía de Usuario: Flujo de Captura – Estudios de Retiro

Esta guía describe la pantalla de inicio del proceso para capturar un estudio de retiro dentro de la aplicación Actividad PRO.

## Descripción de la Pantalla

La pantalla es una portada o carátula que sirve como introducción al flujo de trabajo. 

- **Fondo:** Predominantemente blanco, con elementos gráficos de diseño en color azul en el lateral izquierdo y esquinas.
- **Logotipos:** Se muestra el logotipo de "ACTIVIDAD PRO" de forma prominente en el centro-izquierda, y debajo, el logotipo de "Profuturo 25 AÑOS".
- **Texto Principal:** En el centro de la pantalla se lee el título: "Flujo de captura — estudios de retiro".

Esta pantalla no contiene campos de formulario, botones de acción ni elementos interactivos. Su única función es informar al usuario que ha iniciado el proceso correcto.

## Pasos a Seguir

Dado que esta es una pantalla informativa, se asume que ya has realizado pasos previos para llegar aquí (por ejemplo, seleccionar un cliente y elegir la opción de 'Estudio de Retiro').

1.  **Confirmación Visual:** Verifica que el título "Flujo de captura — estudios de retiro" corresponde a la tarea que deseas realizar.
2.  **Esperar a la Siguiente Pantalla:** La aplicación debería avanzar automáticamente a la primera pantalla de captura de datos después de unos segundos.
3.  **Interacción (Si aplica):** Si la aplicación no avanza sola, busca un botón como "Continuar", "Siguiente" o "Iniciar" que podría no ser visible en esta imagen de ejemplo, o simplemente desliza la pantalla si el flujo lo permite.

## Posibles Errores y Soluciones

- **Error:** La aplicación se queda congelada en esta pantalla y no avanza.
  - **Solución:** Intenta cerrar forzosamente la aplicación y volver a abrirla. Si el problema persiste, reinicia el dispositivo. Como último recurso, reinstala la aplicación o contacta a soporte técnico.

- **Error:** Llegaste a esta pantalla por equivocación.
  - **Solución:** Utiliza el botón o gesto de "Atrás" de tu dispositivo para regresar al menú anterior y seleccionar la opción correcta.

# Guía: Funcionalidades del Estudio Digital de Retiro (EDR)

Esta guía describe la pantalla informativa sobre las funcionalidades y beneficios del Estudio Digital de Retiro (EDR) dentro de la aplicación Actividad Pro.

## Descripción de la Pantalla

La imagen es una diapositiva informativa, no una pantalla interactiva de la aplicación. Presenta un resumen gráfico de las características de la herramienta "Estudio Digital de Retiro".

- **Título Principal:** "Estudios Digital de Retiro – Funcionalidades y beneficios".
- **Icono Central:** Una calculadora estilizada con las siglas "EDR PRO".
- **Columnas de Información:** Alrededor del icono central, se listan diversas funcionalidades con sus respectivos iconos:
    - **Estimación de pensión:** Menciona el uso de metodología Profuturo y variables AMAFORE.
    - **Disponibilidad:** 24/7 para Contacto, Cita o cierre, sin límite de envíos.
    - **Proyección personalizada:** Ofrece 12 tipos de estudios (IMSS, ISSSTE, Patronal, Mixto).
    - **Captura intuitiva:** Alertas visuales y previsualización del estudio.
    - **Calculadora de Apovol:** Muestra el impacto del ahorro voluntario en la pensión.
    - **Notificación de envío:** Respaldo por correo para el asesor y un historial de envíos.
    - **Direcciona la asesoría:** Indicadores para guiar la conversación con el cliente.
    - **Conoce tu futuro:** Clasifica los posibles resultados de la pensión (Pensión, Garantizada, Negativa de pensión).

## Contexto

Esta pantalla aparece como parte del flujo para generar un Estudio Digital de Retiro. Es probable que se muestre como un resumen o introducción a la herramienta después de haber seleccionado al cliente o haber iniciado el proceso de asesoría.

## Pasos a Seguir / Interpretación

Dado que es una pantalla informativa, los pasos se centran en comprender la información para utilizar la herramienta eficazmente.

1.  **Revisa las Capacidades:** Familiarízate con todas las funcionalidades que ofrece la herramienta EDR. Esto te permitirá explicarle mejor los beneficios al cliente.
2.  **Identifica el Tipo de Estudio:** Ten presente los 12 tipos de proyecciones disponibles para seleccionar la que corresponda al régimen del cliente (IMSS L73/L97, ISSSTE, etc.).
3.  **Utiliza las Herramientas Adicionales:** Recuerda que cuentas con la "Calculadora de Apovol" para mostrar escenarios de ahorro voluntario y la sección "Conoce tu futuro" para explicar el resultado final de la proyección.
4.  **Confirma el Envío:** Una vez generado y enviado el estudio, ten la certeza de que recibirás una notificación por correo como respaldo y podrás consultar el historial en la app.
5.  **Procede a la Generación:** Después de revisar esta información, el siguiente paso en la aplicación sería proceder a la pantalla de captura de datos para generar el estudio.

## Posibles Errores o Dudas Comunes

- **No se envía la notificación por correo:** Verificar la conexión a internet y la configuración de correo del asesor. Si el problema persiste, puede ser un retraso en el servicio.
- **La proyección arroja un resultado inesperado:** Asegurarse de que todos los datos capturados (semanas cotizadas, salario, etc.) sean correctos. Un dato erróneo alterará todo el cálculo.
- **No encuentro el tipo de estudio específico:** Revisar si el régimen del cliente está entre las 12 opciones disponibles. Si no es así, es posible que ese caso particular no esté cubierto por la herramienta.
- **La Calculadora de Apovol no se muestra:** Podría ser una funcionalidad que se activa solo bajo ciertas condiciones o tipos de estudio. Verificar si se cumplen los requisitos.

# Guía para Generar un Estudio de Retiro

## Descripción de la Pantalla

Esta pantalla es el paso "Estudio de Retiro" dentro del flujo de la aplicación Actividad Pro. Se presenta después de haber seleccionado un cliente o prospecto. En la parte superior, se muestra un resumen con los datos del cliente (nombre, CURP, correo y celular) para su verificación.

El área principal, titulada "Tipo de estudio", muestra cuatro opciones en recuadros para que selecciones el régimen de retiro correspondiente:
- **IMSS**
- **ISSSTE**
- **MIXTO**
- **PATRONAL**

En la parte inferior se encuentra el botón "NUEVO ESTUDIO" para iniciar el cálculo.

## Pasos a Seguir

Este proceso es la continuación de la selección de un cliente desde los módulos de "Contacto", "Citas" o "Cierres".

1.  **Verificar Datos del Cliente**: Confirma que los datos (nombre, CURP) que aparecen en la parte superior de la pantalla corresponden al cliente para el cual realizarás el estudio.
2.  **Seleccionar el Tipo de Estudio**: Toca el recuadro que corresponda al régimen de retiro de tu cliente. Las opciones disponibles son:
    *   **IMSS**: Para trabajadores bajo Ley 73 o Ley 97.
    *   **ISSSTE**: Para trabajadores del estado (Anticipado, Cesantía y Vejez).
    *   **PATRONAL**: Para conceptos 107, 108 y 111.
    *   **MIXTO**: Para casos que combinan IMSS Ley 97 con los 3 tipos de estudios ISSSTE.
3.  **Iniciar el Estudio**: Una vez seleccionado el régimen correcto, presiona el botón "NUEVO ESTUDIO" para que la aplicación realice el cálculo y genere el resultado.

## Posibles Errores y Resultados

*   **Selección de Régimen Incorrecto**: Elegir un tipo de estudio que no corresponde al historial de cotización del cliente resultará en un cálculo erróneo o en un mensaje de "Negativa de Pensión". Es fundamental validar el régimen del cliente antes de este paso.
*   **Resultado: Pensiones Garantizadas**: Si el sistema determina que los saldos del cliente son insuficientes pero sí cumple con el tiempo mínimo de cotización, la aplicación mostrará este resultado en lugar de un monto de pensión calculado.
*   **Resultado: Negativa de Pensión**: Este mensaje aparecerá si el cliente no alcanza el tiempo mínimo de cotización requerido para su régimen. Esto no es un error de la aplicación, sino el resultado del análisis de su situación.

# Guía para Generar un Estudio de Retiro

Esta guía detalla los pasos 3 y 4 del flujo de captura dentro de la aplicación Actividad Pro para generar y enviar un estudio de retiro a un prospecto.

## Descripción de la Pantalla

El proceso se divide en dos pantallas principales:

1.  **Datos de la cuenta mixta IMSS:** Un formulario para ingresar los datos laborales y de ahorro del cliente, separados en secciones para IMSS e ISSSTE. Los campos obligatorios están marcados con un asterisco (`*`). En la parte inferior se encuentra el botón "CONTINUAR".
2.  **Estudio para el Retiro (Vista Previa):** Una visualización del documento final que se generará. Muestra un resumen de los datos personales y las proyecciones de pensión. Contiene los botones "CORREGIR" y "ENVIAR".

## Pasos a Seguir

**Contexto:** Has llegado a esta pantalla después de completar los pasos de 'Acceso', 'Contacto', 'Cita' y 'Cierre' del flujo de captura.

1.  **Completar Datos:** En la pantalla "Datos de la cuenta mixta IMSS", llena todos los campos obligatorios marcados con un asterisco (`*`). Asegúrate de que la información sea correcta y precisa.
2.  **Ahorro Voluntario:** Activa o desactiva la opción de "Ahorro voluntario mensual" según corresponda y especifica el monto si está activo.
3.  **Continuar:** Una vez que todos los datos estén completos, presiona el botón azul **"CONTINUAR"**.
4.  **Visualizar Previo:** La aplicación te mostrará una vista previa del "Estudio para el Retiro".
5.  **Revisar Información:** Verifica cuidadosamente que todos los datos en el documento (nombre, CURP, montos, proyecciones, etc.) sean correctos.
6.  **Realizar Correcciones (si es necesario):** Si encuentras algún error, presiona el botón **"CORREGIR"**. Serás redirigido a la pantalla anterior para editar la información.
7.  **Enviar Estudio:** Si toda la información es correcta, presiona el botón **"ENVIAR"**. El estudio se enviará al correo electrónico de tu prospecto y tú recibirás una copia de confirmación.

## Posibles Errores y Soluciones

*   **Error: "Este campo es obligatorio"**: 
    *   **Causa:** No se ha llenado uno o más campos marcados con un asterisco.
    *   **Solución:** Revisa todo el formulario y completa la información faltante antes de presionar "CONTINUAR".
*   **Error: El botón "CONTINUAR" no avanza**: 
    *   **Causa:** Puede haber datos en un formato incorrecto (ej. letras en campos numéricos) o campos obligatorios sin llenar.
    *   **Solución:** Verifica que todos los datos estén en el formato adecuado y que no falte información requerida.
*   **Error: Falla al enviar el estudio**:
    *   **Causa:** Posiblemente una falla en la conexión a internet.
    *   **Solución:** Asegúrate de tener una conexión estable (Wi-Fi o datos móviles) e intenta presionar "ENVIAR" nuevamente.

# Guía: Validación del Estudio de Retiro

Este documento explica el paso "Estudio de Retiro" dentro del flujo de captura de la aplicación Actividad Pro, que consiste en la generación y validación de un documento PDF con la proyección de pensión para el cliente.

## Descripción de la Pantalla

La pantalla muestra una vista previa del documento "Estudio para el Retiro" que se genera automáticamente. En la parte superior, un diagrama de flujo indica que nos encontramos en el paso **"Estudio de retiro"**, posterior a "Cierre" y anterior a "Indicadores".

El documento generado se compone de tres secciones principales:

1.  **Portada:** Contiene el logo de Profuturo, el título "ESTUDIO PARA EL RETIRO", el régimen (ej. IMSS LSS97) y datos de identificación del trámite como el nombre del trabajador, asesor, folio y fecha.
2.  **Detalle del Estudio:** Incluye:
    *   **Datos Personales:** Nombre completo, CURP, RFC, NSS y Saldo en la subcuenta de RCV. Es crucial verificar que estos datos sean correctos.
    *   **Proyección de Seguridad Social:** Una tabla que estima el monto de la pensión bajo diferentes escenarios (Cesantía, Vejez, Riesgo de Trabajo, Invalidez y Vida) y a distintas edades.
    *   **Aportaciones Voluntarias:** Una sección que explica cómo las aportaciones voluntarias pueden incrementar el monto de la pensión, con una tabla de ejemplo mostrando proyecciones con un 5% y 10% de ahorro adicional.
3.  **Página de Contacto:** Muestra información de contacto de Profuturo, un código QR para descargar la aplicación Profuturo Móvil y enlaces a redes sociales.

## Contexto y Pasos para el Usuario

Este paso es la culminación de la captura de datos del cliente. El sistema utiliza toda la información recopilada para generar un documento formal que el asesor presentará al trabajador.

**Instrucciones:**

1.  **Revisión del Documento:** Al llegar a esta pantalla, el sistema ya habrá generado el PDF. Tu primera acción es revisar detenidamente todo el contenido.
2.  **Validar Datos Personales:** Confirma que el nombre, CURP, NSS y RFC del cliente sean exactamente los correctos. Un error aquí invalida todo el estudio.
3.  **Analizar Proyecciones:** Revisa las tablas de proyecciones para asegurar que los cálculos parezcan lógicos y consistentes con los datos de entrada (ej. Saldo RCV).
4.  **Presentar al Cliente:** Utiliza este documento como herramienta principal para explicarle al cliente sus opciones de retiro, el impacto de las semanas cotizadas y los beneficios de realizar aportaciones voluntarias.
5.  **Continuar al Siguiente Paso:** Una vez que el estudio ha sido revisado y presentado, procede al último paso del flujo, **"Indicadores"**, para finalizar el registro de la actividad.

## Posibles Errores y Soluciones

*   **Error:** Los datos personales en el PDF son incorrectos.
    *   **Causa:** Error de captura en los pasos iniciales ("Acceso" o "Contacto").
    *   **Solución:** Debes regresar a los pasos anteriores del flujo para corregir la información del cliente. Una vez corregida, el estudio deberá generarse de nuevo. No es posible editar el PDF directamente en esta pantalla.

*   **Error:** El documento PDF no se genera o muestra un error de visualización.
    *   **Causa:** Posible problema de conexión a internet, un error temporal del sistema o del navegador.
    *   **Solución:** Verifica tu conexión a internet. Intenta recargar la página o reiniciar el flujo para ese cliente. Si el problema persiste, contacta a la mesa de ayuda de TI, ya que podría ser un fallo del sistema.

*   **Error:** Las proyecciones de pensión son cero o parecen ilógicas.
    *   **Causa:** Puede deberse a que los datos base (como el Saldo en la cuenta o las semanas cotizadas) son incorrectos o nulos. También podría ser un error de cálculo del sistema.
    *   **Solución:** Valida los datos económicos y de cotización que se ingresaron en los pasos previos. Si los datos de entrada son correctos pero el cálculo es erróneo, repórtalo inmediatamente a soporte técnico.
